POS Order Type

This apps helps you to set order type on pos order



App Features
  • Create Order Type
  • Set order types for every pos
  • Set default order type for each pos and system will set by default on every pos order
  • Print Order type on pos receipt
  • Enable/Disable order type for each pos
  • PoS report group by order type
  • Compatible with community and enterprise
